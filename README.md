# UdobniySoftExcelApp
Тестовое задание дял компании Удобный Софт на позицию Java Developer.

Для запуска приложения достаточно скачать каталог FinishedApp со всем содержимым и запустить скрипт-файл "start.bat". После чего в браузере будет доступен swagger по адрессу 
" localhost:8080/swagger-ui/index.html#/ ".
